Deploy App

In order to deploy your app, you must make a code change.

You must update the manifest.yml file in your source code repository, this will automatically trigger the delivery pipeline. You can copy & paste the code below into the manifest.yml file in your repository.

---
# We have created a temporary app in the gallery
# We have created a delivery pipeline
applications:
# We have reserved a name/route for your app
- name: {{ wizardCtrl.options.userInput.name }}
  host: {{ wizardCtrl.options.userInput.host }}
  domain: {{ wizardCtrl.options.userInput.domain.entity.name }}
  # We have provisioned and bound services
  services:
  - {{ service.name }}
