Use the Cloud Foundry CLI to execute commands directly against a Helion Cloud Foundry.


Getting Started

Before starting follow these steps

1. Download and install the CLI - see here

2. Set the api endpointcf api {{context.api || '[Cloud Foundry API endpoint]'}}

3. Login$ cf login API endpoint: {{context.api || '[Cloud Foundry API endpoint]'}} Email> {{context.username || '[username]'}} Password> [password] Authenticating... OK

4. Target an org and spacecf target -o {{context.orgName || '[organization name]'}} -s {{context.spaceName || '[space name]'}}


Organization Commands

List
To list all orgscf orgs
Rename
To rename an orgcf rename-org {{context.orgName || '[organization name]'}} [name]
Information
To fetch org informationcf org {{context.orgName || '[organization name]'}}
Users
List each org role's userscf org-users {{context.orgName || '[organization name]'}}

Space Commands

List
To list the spaces in the targeted orgcf spaces
Rename
To rename a spacecf rename-space {{context.spaceName || '[space name]'}} [name]
Information
To fetch space informationcf space {{context.spaceName || '[space name]'}}
Users
List each space role's userscf space-users {{context.orgName || '[organization name]'}} {{context.spaceName || '[space name]'}}
