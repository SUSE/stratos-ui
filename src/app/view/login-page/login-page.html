Helion Stackato Web Console

Use this Web console to develop, compose, and manage Cloud Native workloads. This interface enables you to perform common management tasks, in an integrated experience.

Develop

Use this Web Console to create and deploy applications that are designed for Helion Cloud Foundry. You'll be able to set up a delivery pipeline that automates your build, test, and deployment cycles.

Compose

You will find application services that you can bind to your Helion Cloud Foundry apps. Use this Web Console to enable storage, messaging, and data processing services.

Manage

As a Helion Stackato admin, you'll be able to register and manage the platform services your development teams rely on.
